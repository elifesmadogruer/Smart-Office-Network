# Smart Office Network

Smart Office network simulation developed using Cisco Packet Tracer.

## Project Overview
This project simulates a smart office environment with routers, servers, PCs, and IoT devices.

## Features
- Router configuration
- DHCP services
- DNS services
- IoT device connectivity
- Web-based monitoring and control
- Network communication between office components

## Technologies
- Cisco Packet Tracer
- Networking
- IoT
